# termes

termes - небольшой помошник в общение. С его помощью можно общаться, и не волноваться за безопастность ваших сообщений, тк. используется шифрование (многослойное, E2E)
| Уровень | Протокол / Алгоритм                              | Назначение                                                  |
|:--------|:--------------------------------------------------|:------------------------------------------------------------|
| **1**   | **TLS 1.3** (RSA-4096 сертификат)                | Шифрование всего трафика клиент ↔ сервер                    |
| **2**   | **X25519 ECDH**                                  | Обмен эфемерным ключом между клиентами                     |
| **3**   | **HKDF-SHA256**                                  | Деривация 256-битного симметричного ключа                  |
| **4**   | **AES-256-GCM** (96-bit random nonce)            | Шифрование и аутентификация каждого сообщения              |

В будущем планирую развивать данное приложение, и сделать визуал. Клиент работает на линуксе, на остальнных клиентах не проверял.
